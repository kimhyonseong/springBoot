  - 임시 디렉토리 / 실제 사용 이미지 디렉토리 구분(완료)
  - 삽입(완료)
  - 수정(완료)

  - 음식 평가(완료) - 테스트 케이스 만들어야됨
  - 음식 구매
    - 즉시 구매(1개 구매)(진행중)

      - 일정 시간 지났는데 결제 완료되지 않으면 임시 테이블에서 수량 삭제
      - 구매 내역 / 구매 아이템 DB 저장(진행중)
  - 장바구니
    - 넣기(완료)
